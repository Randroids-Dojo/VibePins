Game-Loop Orchestrator (Frame/Ball State Machine)

Branch

PR

Changed
Added the pure frame/ball state machine that turns the isolated systems into a game (the spine of F-008). New framework-free module src/game.ts exposing the Game class: drive it with recordBall(pinsDowned) once per settled ball and it returns a BallResult with the outcome (await-ball, frame-reset, game-over), the reset to run (between-balls lifts only fallen pins and leaves standing pins for the next throw per REQ-009; rerack re-racks all ten at frame end per REQ-010), the live pinsStanding, and the live GameScore. The game ends after the tenth and exposes summary() with the final score (REQ-011). A finished non-tenth frame opens its successor lazily, so no trailing empty frame ever reaches the validator.
